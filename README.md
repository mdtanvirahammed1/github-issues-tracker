1.What is the difference between var, let, and const?
Think of these as boxes to hold your data.
var: An old way to make a box. It is a bit messy, so we try not to use it much now.
let: A box where you can change the item inside later.
const: A box where you put an item once, and it is locked forever. You cannot change it.

2. What is the spread operator (...)?
Imagine you have a bag of apples and you want to empty them into a new basket. The ... operator takes everything out of the old bag and "spreads" it into the new one.It is a quick way to copy or combine lists (arrays) of data.

3. What is the difference between map(), filter(), and forEach()?
map(): Creates a new array by applying a function to every element.
filter(): Creates a new array with elements that pass a specific condition.
forEach(): Executes a provided function once for each array element without returning a new array.

4.What is an arrow function?
This is just a shorter, faster way to write a function (a small task).
Old way: function sayHello() { return "Hi"; }
Arrow way: const sayHello = () => "Hi";
It saves space and looks cleaner.

5. What are template literals?
Before, joining text and data was hard with + signs. Now, we use backticks (the key above the Tab key: `).
Example: `Hello, ${name}!`
It makes it very easy to put data inside a sentence without messy symbols.
